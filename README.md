# VANI Premium Dashboard

**VANI** stands for **Voice Assisted Neural Intelligence** — an AI-powered data analytics assistant built with Python and Streamlit.

VANI helps users upload Excel files and instantly generate insights through an interactive chatbot, smart dashboards, charts, filters, and voice-enabled responses.

---

## Live Features

✅ Secure Login Authentication
✅ Multi-file Excel Upload (.xlsx / .xls)
✅ AI Chatbot for Data Queries
✅ Smart Prompt Suggestions
✅ Dynamic Charts (Line, Bar, Histogram, Scatter, Pie, Area)
✅ Sidebar Filters / Slicers
✅ Live KPI Metrics
✅ Browser Text-to-Speech Output
✅ Download Filtered CSV Reports
✅ Premium Responsive UI
