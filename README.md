# VANI Premium Dashboard

**VANI** stands for **Voice Assisted Neural Intelligence** — an AI-powered data analytics assistant built with Python and Streamlit.

VANI helps users upload Excel files and instantly generate insights through an interactive chatbot, smart dashboards, charts, filters, and voice-enabled responses.

---

## Live Features

✅ Secure Login Authentication\n
✅ Multi-file Excel Upload (.xlsx / .xls)\n
✅ AI Chatbot for Data Queries\n
✅ Smart Prompt Suggestions\n
✅ Dynamic Charts (Line, Bar, Histogram, Scatter, Pie, Area)\n
✅ Sidebar Filters / Slicers\n
✅ Live KPI Metrics\n
✅ Browser Text-to-Speech Output\n
✅ Download Filtered CSV Reports\n
✅ Premium Responsive UI\n
